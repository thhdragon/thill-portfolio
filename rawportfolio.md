* **Reflection:**"This was my hardest project, but it perfectly bridged my independent programming with my Network Essentials class. Instead of just reading about IP headers and ICMP, I actively parsed the packet bits to understand how hops work. Building the DNS library made me realize 'reversing' an IP for a lookup means flipping the octets rather than the whole string. Implementing UDP port 53 in code cemented these networking concepts far better than a textbook ever could."
